No:ER-LDRWI001 20200331

Error Message

Key Word:

Realtek,WiFi ,wpa_supplicant_hostapd

error message show:

fatal error: netlink/genl/genl.h: No such file or directory

Directions

Compiler not found the libnl-3-dev.

The WPA Supplicant hostapd need to used libnl library to compiler.

Countermeasures

download the libnl-3-dev and install.

type

sudo apt-get install libnl-3-dev
sudo apt-get install libnl-genl-3-dev
And need to modify the .config file to add
CONFIG_LIBNL32=y

Download link

http://www.infradead.org/~tgr/libnl/files/libnl-3.2.25.tar.gz
