No:ER-LA0003 2018-03-06

Error Message

Key Word: Android 7.0 , Linux

error message show:

Communication error with Jack server (58)

Countermeasures

This message is only use the TCP port, So need to change "jack" TCP port setting

To chang tow files  .jack-settings and .jack-server/config.properties 

HOME# vim .jack-settings

# Server settings
SERVER_HOST=127.0.0.1
SERVER_PORT_SERVICE=8076
SERVER_PORT_ADMIN=8077
#change to
SERVER_PORT_SERVICE=8086
SERVER_PORT_ADMIN=8087

# Internal, do not touch
SETTING_VERSION=4
and
HOME# vim .jack-server/config.properties

#Tue Mar 06 14:41:07 CST 2018
jack.server.max-jars-size=104857600
jack.server.max-service=4
jack.server.service.port=8076
jack.server.max-service.by-mem=1\=2147483648\:2\=3221225472\:3\=4294967296
jack.server.admin.port=8077
jack.server.config.version=2
jack.server.time-out=7200

#change to
jack.server.service.port=8086
jack.server.admin.port=8087
and

chmod 600 .jack-server/config.properties
