How to Read Write Switch Register 

Key Word:

Realtek Switch ; phy register

Directions

used rtl_SDK in path type

echo [command] [phyID] [regAdd] (Data) > /proc/rtl865x/phyReg

[command]
8367phyr       # Read phy address data
8367phyw     # Write data phy address data
8367test       # test mode

[phyID]
Is LAN port number

[phyAdd]
Is pht register address

The C file in
./rtl819x/linux-2.6.30/drivers/net/rtl819x/rtl865x_proc_debug.c
